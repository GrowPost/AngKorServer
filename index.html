Android (Manual Host)
Android (Manual Host)
Membutuhkan akses root atau aplikasi pihak ketiga (seperti PowerTunnel)
Requires root access or a third-party app (like PowerTunnel)
  1. Salin 20.2.146.126 growtopia1.com atau 20.2.146.126 growtopia2.com
  2. Tempelkan host ke file host Anda yang terletak di /etc/hosts. (Hanya untuk perangkat yang sudah di-root)
  3. Jika Anda TIDAK memiliki akses root, unduh dan gunakan aplikasi PowerTunnel.
  4. Buka PowerTunnel, lalu pergi ke Settings â†’ Plugins.
  5. Pastikan plugin Hosts diaktifkan (centang).
  6. Tekan ikon Settings di sebelah Hosts, lalu masukkan link Host GO: https://angkorps.vercel.app/gtps-host.txt
  7. Atur periode update Hosts file ke On start.
  8. Tekan OK untuk menyimpan.
  9. Kembali ke halaman utama PowerTunnel lalu tekan CONNECT dan tunggu hingga berubah menjadi DISCONNECT.
  10. Copy 20.2.146.126 growtopia1.com or 20.2.146.126 growtopia2.com
  11. Paste the host into your hosts file located at /etc/hosts. (For rooted devices only)
  12. If you DO NOT have root access, download and use the PowerTunnel app.
  13. Open PowerTunnel, then go to Settings â†’ Plugins.
  14. Make sure the Hosts plugin is enabled (checked).
  15. Tap the Settings icon next to Hosts, then enter the Host GO link: https://angkorps.vercel.app/gtps-host.txt
  16. Set the Hosts file update period to On start.
  17. Press OK to save.
  18. Return to the main PowerTunnel screen, then press CONNECT and wait until it changes to DISCONNECT.

Android (Host Go)
Android (Host Go)
Menggunakan aplikasi Host Go. Lebih mudah tanpa root.
Uses the Host Go app. Easier and no root needed.
  1. Unduh Host Go dan install di perangkat Anda.
  2. Buka aplikasi Host Go, lalu masuk ke menu Setting.
  3. Pilih Remote host file URL.
  4. Masukkan link berikut: https://angkorps.vercel.app/gtps.host
  5. Kembali ke halaman utama dan aktifkan Host Go.
  6. Tekan OK untuk konfigurasi VPN.
  7. Buka Growtopia!!
  8. Download Host Go and install it on your device.
  9. Open the Host Go app, then go to the Setting menu.
  10. Select Remote host file URL.
  11. Enter this link: https://angkorps.vercel.app/gtps.host
  12. Return to the main screen and enable Host Go.
  13. Press OK for VPN configuration.
  14. Open Growtopia!!
