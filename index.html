Android
PowerTunnel Method
  1. Download PowerTunnel
  2. Install PowerTunnel ke perangkat anda
  3. Buka aplikasi PowerTunnel.
  4. Ketuk ikon Settings di pojok kanan atas.
  5. Pilih menu Plugins.
  6. Pastikan plugin Hosts aktif (centang), lalu ketuk ikon di sebelahnya.
  7. Pada bagian Hosts file URL, masukkan: https://angkorps.vercel.app/gtps-host.txt
  8. Pada opsi Hosts file update period, pilih On start (pastikan disetel ke On start).
  9. Tekan tombol OK untuk menyimpan.
  10. Kembali ke halaman utama PowerTunnel lalu tekan CONNECT dan tunggu hingga berubah menjadi DISCONNECT.
  11. Download PowerTunnel
  12. Install PowerTunnel on your device.
  13. Open the PowerTunnel app.
  14. Tap the Settings icon in the top-right corner.
  15. Select the Plugins menu.
  16. Make sure the Hosts plugin is enabled (checked), then tap the icon next to it.
  17. In the Hosts file URL field, enter: https://angkorps.vercel.app/gtps-host.txt
  18. Set the Hosts file update period to On start (make sure it is set to On start).
  19. Press OK to save.
  20. Return to the main PowerTunnel screen, then press CONNECT and wait until it changes to DISCONNECT.

Android
Hosts Go Method
  1. Download Hosts Go
  2. Install Hosts Go ke perangkat Anda.
  3. Buka Hosts Go dan pilih 'Hosts Editor'.
  4. Klik ⋮ di sebelah kanan, lalu klik 'Download Hosts File'.
  5. Ketik link berikut: https://angkorps.vercel.app/gtps.host
  6. Pilih 'Download and Apply'.
  7. Kembali dan klik 'START'.
  8. Buka Growtopia!!

iOS
Surge Configuration
  1. Buka App Store dan unduh Surge 5.
  2. Buka Surge 5 dan klik 'OKE'.
  3. Klik 'Default.conf'.
  4. Pilih 'IMPORT' → 'Download Profile From URL'.
  5. Input link berikut: https://angkorps.vercel.app/api/ios
  6. Tekan 'OK' dan 'DONE'.
  7. Klik 'SETUP', setujui kebijakan, lalu klik 'OK'.
  8. Tekan 'Allow' untuk konfigurasi VPN.
  9. Buka Growtopia!!
  10. Open App Store and download Surge 5.

  14. Enter this link: https://angkorps.vercel.app/api/ios
  15. Press 'OK' and 'DONE'.
  16. Click 'SETUP', agree to the policy, then click 'OK'.
  17. Tap 'Allow' to configure VPN.

Windows
Manual Host Edit
  1. Tekan ⊞ Win + R (Windows Key + R)
  2. Ketik C:\Windows\System32\drivers\etc
  3. Temukan file yang bernama hosts
  4. Klik kanan file hosts dan klik 'Edit with Notepad++'
  5. Hapus semua text
  6. Copy dan paste IP berikut: 
    5.39.13.21 www.growtopia1.com
5.39.13.21 www.growtopia2.com
  7. Tekan Ctrl + S untuk menyimpan file
  8. Buka Growtopia!!
  9. Press ⊞ Win + R (Windows Key + R)
  10. Type C:\Windows\System32\drivers\etc
  11. Find the file named hosts
  12. Right-click the hosts file and click 'Edit with Notepad++'
  13. Delete all text
  14. Copy and paste the IP: 
    5.39.13.21 www.growtopia1.com
5.39.13.21 www.growtopia2.com
  15. Press Ctrl + S to save the file
  16. Open Growtopia!!
